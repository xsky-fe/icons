# Wizard icons
This package provides [Wizard icons](https://xsky-fe.github.io/icons/) as a set of React Components.

## Features
* React 16.8 +
* Typescript 3.5+
* Webpack 4
* Babel 7
* Hot Module Replacement

## Npm cli
```bash
# svg to react and create index
yarn build-react
# ts to js
yarn ts2js
# development
yarn start
# build
yarn build
# deploy
yarn deploy
```
